# Coding conventions

## Naming Conventions
- **Variables:** camelCase: e.g. helloWorld
- **Constants:** capitalize: e.g. HELLOWORLD
- **Functions:** camelCase: e.g. helloWorld
- **Classes:** CamelCase: e.g. HelloWorld

## Comments
- less is more

## Design Approach
- Bottom-up Approach
- one class per file

## Imports
- at the top of the file
- import numpy as np
